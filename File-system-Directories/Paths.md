##############Relative and Absolute Paths##############

Absolute Path:

It goes from root folder to the desired destination.
example:
/Work/Project/Python/snake.py

Relative Path:

Relative path is for example we are in the current working
Directory "Python" then we can access the path of snake.py which is the relative path ,,through current director we are in 

example: ./snake.py
>>>> ./ means look in the current folder this file

>>> IMPORTANT POINTS:

KEY DIFFERENCE BETWEEN ABSOLUTE AND RELATIVE PATHS>>

"The Absoulte Path is always relative to the Root of the Computer such as disk drive,C"

"The Relative Path is always relative to the Current Working Directory/Folder "
